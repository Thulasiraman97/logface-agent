# Logface Agent

MongoDB attendance query agent for OpenClaw AI assistant.

## Local Setup
npm install
node mongo-query.js attendance_today

## Query Examples
node mongo-query.js attendance_today
node mongo-query.js attendance_date 15-03-2026
node mongo-query.js total_employees
node mongo-query.js employee_search Ravi
node mongo-query.js pending_leaves
node mongo-query.js monthly_summary 03-2026
